# 3.2 The 4 Types of Moments

## Core Framework
Identifying the four types of moments to focus on for maximum impact.

## The 4 Moment Types
1. **Peak Moments**: High emotional intensity experiences
2. **Pit Moments**: Low emotional intensity experiences
3. **Transition Moments**: Moving between journey stages
4. **End Moments**: How experiences conclude

## Key Learning Points
- **Peak Moment Creation**: Designing experiences that create positive emotional peaks
- **Pit Moment Prevention**: Avoiding or minimizing negative experiences
- **Transition Optimization**: Making stage changes smooth and positive
- **End Moment Impact**: Ensuring experiences end on a positive note

## Practical Applications
- **Moment Identification**: Finding each type of moment in your journey
- **Impact Assessment**: Measuring the psychological impact of each moment
- **Improvement Prioritization**: Focusing on highest-impact moments
- **Design Strategy**: Creating specific strategies for each moment type

## Exercises
1. **Moment Classification**: Classify each touchpoint by moment type
2. **Impact Measurement**: Assess the psychological impact of each moment
3. **Improvement Planning**: Create specific plans for each moment type

## Key Takeaways
- Peak moments create lasting positive memories
- Pit moments can destroy user trust and satisfaction
- Transition moments are often overlooked but crucial
- End moments determine how users remember the entire experience
