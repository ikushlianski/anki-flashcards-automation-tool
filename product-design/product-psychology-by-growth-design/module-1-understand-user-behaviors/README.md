# Module 1: Understand User Behaviors
## Week 1 - Foundation of Product Psychology

### Learning Objectives
- Master storytelling psychology to build user empathy
- Learn behavioral mapping to motivate ethical actions
- Apply the Psych Framework to represent user behaviors
- Identify gaps in user experience through proven mental models

---

## Submodules

### [1.1 Storytelling Psychology](./1.1-storytelling-psychology.md)
Using the psychology behind stories to build user empathy and find experience gaps.

### [1.2 Behavioral Mapping](./1.2-behavioral-mapping.md)
Motivating people to act ethically while understanding what impacts their ability to take action.

### [1.3 Psych Framework](./1.3-psych-framework.md)
Advanced applications of motivations and abilities to represent user behaviors systematically.

### [1.4 GEQs (3 Questions Framework)](./1.4-geqs-3-questions-framework.md)
The 3 exact questions to uncover deep customer needs and motivations.

### [1.5 Visual Design Psychology](./1.5-visual-design-psychology.md)
Using visual design principles to enhance psychological impact and create more effective user experiences.

### [1.6 Information Architecture Psychology](./1.6-information-architecture-psychology.md)
Organizing information and content in ways that align with user mental models and psychological processing patterns.
