# 1.4 GEQs (3 Questions Framework)

## Core Concept
The 3 exact questions to uncover deep customer needs and motivations.

## The 3 Questions
1. **What are you trying to accomplish?** (Goal identification)
2. **What's preventing you from doing it?** (Barrier identification)
3. **How do you feel about that?** (Emotional state assessment)

## Key Learning Points
- **Question Sequencing**: Why these questions work in this order
- **Probing Techniques**: How to dig deeper with follow-up questions
- **Pattern Recognition**: Identifying common themes across users
- **Need Prioritization**: Ranking discovered needs by importance

## Practical Applications
- **User Interview Framework**: Structured approach to user research
- **Need Discovery**: Uncovering hidden user requirements
- **Emotional Mapping**: Understanding user feelings and motivations
- **Solution Validation**: Testing if solutions address real needs

## Exercises
1. **GEQ Practice**: Conduct 3 user interviews using the framework
2. **Pattern Analysis**: Identify common themes across interviews
3. **Need Prioritization**: Rank discovered needs by user impact

## Key Takeaways
- These 3 questions uncover the most important user insights
- The sequence matters: goals → barriers → emotions
- Emotional responses often reveal the deepest needs
- Pattern recognition across users reveals universal truths
