# Module 3: Writing Effective Requirements

## Overview
This module teaches you how to write clear, actionable, and comprehensive requirements that drive successful product development. You'll learn frameworks, templates, and best practices for effective requirements writing.

## Learning Objectives
- Master the SMART requirements framework
- Learn user story mapping with AI assistance
- Understand acceptance criteria best practices
- Apply requirements templates and standards
- Write cross-functional requirements effectively

## Topics Covered
1. [The SMART Requirements Framework](./3.1-smart-requirements-framework.md)
2. [User Story Mapping with AI](./3.2-user-story-mapping-ai.md)
3. [Acceptance Criteria Best Practices](./3.3-acceptance-criteria-best-practices.md)
4. [Requirements Templates and Standards](./3.4-requirements-templates-standards.md)
5. [Cross-Functional Requirements](./3.5-cross-functional-requirements.md)

## Key Takeaways
- Clear requirements are the foundation of successful products
- AI can enhance requirements writing and organization
- Templates and standards improve consistency
- Cross-functional collaboration is essential
