# 3.4 Journey Re-ordering for Maximum Psychology

## Core Concept
Re-ordering customer journeys to maximize overall psychological impact.

## Key Learning Points
- **Psychological Flow**: Understanding how experiences build on each other
- **Momentum Building**: Creating experiences that build positive momentum
- **Friction Reduction**: Minimizing barriers between journey stages
- **Value Progression**: Ensuring each stage provides increasing value
- **Emotional Arc**: Creating a positive emotional journey throughout

## Practical Applications
- **Journey Optimization**: Re-ordering touchpoints for better flow
- **Friction Elimination**: Removing barriers between stages
- **Value Sequencing**: Ensuring value increases throughout the journey
- **Emotional Design**: Creating positive emotional arcs

## Exercises
1. **Journey Re-ordering**: Re-arrange your journey for better psychological flow
2. **Friction Audit**: Identify and eliminate journey friction points
3. **Value Progression**: Ensure each stage builds on the previous one

## Key Takeaways
- Journey order significantly impacts psychological outcomes
- Positive momentum builds user confidence and engagement
- Friction between stages can derail entire journeys
- Value progression keeps users engaged and motivated
